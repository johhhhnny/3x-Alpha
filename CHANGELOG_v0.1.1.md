# v0.1.1 更新日志

## 新功能

### 🗨️ GitHub 评论系统
- 集成 **Giscus**，基于 GitHub Discussions 的评论功能
- 支持 GitHub 账户登录
- 完全支持中文显示
- 响应式设计，自适应深色/浅色模式
- 每篇文章自动关联对应的讨论

**配置指南**：详见 `GISCUS_SETUP.md`

### 🔍 搜索功能优化
- 改进搜索结果卡片样式
- 搜索结果现在有分离的卡片视觉设计
- 改善悬停效果和交互反馈
- 优化搜索输入框焦点状态
- 更好的搜索结果可读性

## 改进

- 搜索结果卡片间距优化
- 搜索输入框获得焦点时的视觉反馈
- 搜索清除按钮样式统一
- 结果标题和摘要排版改进

## 需要手动配置

**重要**：为了启用评论功能，需要完成以下配置：

1. 访问 https://giscus.app 获取配置信息
2. 更新 `src/components/elements/giscus-comments.astro` 中的：
   - `data-repo-id`
   - `data-category-id`
3. 确保仓库已启用 Discussions 功能

详细步骤见 `GISCUS_SETUP.md`
